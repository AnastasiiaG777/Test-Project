# Test-Project

Учебный проект на Flask
Простое веб-приложение для создания и просмотра постов.

## Функциональность 

- Главная страница
- Просмотр всех записей
- Добавление новых записей
- Просмотр записей целиком
- Удаление записей

## Использование технологий

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML / Jinja2
- Bootstrap 5
- СSS

## Структура проекта

├── app.py

├── README.md

├── templates/

├── static/
│   ├── css/
│   └── img/


## Запуск проекта

1. Установите зависимость
   
  pip install flask flask-sqlalchemy

   
3. Запустите приложение
   
  python app.py


5. Откройте в браузере:
   
   http://127.0.0.1:5000/
